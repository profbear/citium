how do I...
  1. clone it git clone the https://github.com/profbear/swoopi.git
  2. install deps cd swoopi ; npm i
  3. start it npm start boots a URL like localhost:3000
  4. skim it in the browser at localhost:3000
  5. explore the depths at localhost:3000/explorer

it'll kinda look like this publicly available swoopi.
